1. Yönetici Özeti

Tekur Akademi, 4-18 yaş arası futbol sporcuları için yapılandırılmış antrenman + ölçüm + iletişim platformudur. Şu an mevcut olan tek dosya (index.html, 9.862 satır) içerik referansı / ürün vizyonu mockup'ıdır; gerçek uygulama henüz inşa edilmemiştir. Bu rapor mockup'tan türetilen sistemin ölçeklenebilir, KVKK uyumlu, mobil-öncelikli mimarisini tanımlar.

İçerik Vizyonu (Mockup'ta Tamamlanmış)

  • 5 yaş grubu antrenman programı (U6-U18) × 11 katman
  • 4 alternatif metodoloji (Bozkurt / Aslan / Atilla / Yüce-Ediz)
  • Kayıt + KVKK + sözleşme + sağlık raporu yönetimi (5 adımlı)
  • Antrenör yoklama + puanlama (12 akıllı hata-engelleme)
  • Maç takvimi + Özel Ders Sistemi (sistem önerisi → WhatsApp onay → takvim → sonuç)
  • 6 Çapraz Modül: Sporcu Beslenmesi, Kalecilik, Sezon Dışı Hazırlık, Return to Play, Pozisyon-Özel Drill'ler, Futsal/İç Saha

3.2 Backend

  • API: NestJS (TypeScript). REST + GraphQL hibrit.
  • Auth: SMS OTP + e-posta + biyometrik. RBAC + ABAC.
  • DB: PostgreSQL 16 + TimescaleDB (karne zaman serisi). Redis cache.
  • Dosya: MinIO TR-host (S3 uyumlu).
  • Background jobs: BullMQ — bildirim, KVKK otomatik silme, karne PDF.
  • Real-time: Socket.io — yoklama anlık güncelleme.

5. KVKK + Güvenlik Mimarisi (Kritik)

5.1 Şifreleme

  • At-rest: AES-256, ayrı KMS. Özel nitelikli (sağlık + biyometrik) ayrı anahtar.
  • In-transit: TLS 1.3, HSTS, certificate pinning (mobil).
  • DB: PostgreSQL Row-Level Security + column-level encryption (sağlık).

5.2 Erişim Kontrolü

  • RBAC + ABAC (örn. "Diyetisyen sadece kendisine atanan sporcuları görür").
  • Audit log: her sağlık verisi erişimi → 5 yıl saklanır.
  • 2FA zorunlu: tüm uzman + admin.

5.4 KVKK Hak Talep Akışı (Otomatize)

  • Veli/sporcu mobil app'ten "Verilerimi gör/sil/düzelt" butonu
  • 30 gün içinde otomatik cevap (kanun gereği)
  • Talep tipi: bilgi (otomatik PDF) / silme (admin onaylı) / düzeltme (form)

5.5 Yurt Dışı Aktarımı

YOK → tüm hosting Türkiye'de. Üçüncü parti seçimi TR-host kontrolü ile (örn. SendGrid yerine TR mail servisi).

5.6 Güvenlik Denetimleri

  • Yıllık penetration test (3. parti)
  • OWASP Top 10 + Mobil OWASP Top 10
  • Bağımlılık tarama (Snyk/Dependabot, sürekli)
  • ISO 27001 hedef (yıl 2)

6. Yol Haritası (Faz Bazlı)

Faz 1 — MVP (3-4 ay)

  • Kayıt + KVKK rıza akışı (5 adım)
  • Veli + Antrenör + Admin auth (SMS OTP)
  • Antrenör yoklama + puanlama (offline-first)
  • Aylık karne (tek yaş grubu pilot, U10)
  • WhatsApp bildirim entegrasyonu
  • Ödeme (aidat — iyzico)

Faz 2 — Genişleme (2-3 ay)

  • 5 yaş grubu programının tam içerik aktarımı (CMS)
  • Maç sistemi + lineup + post-match puanlama
  • Özel ders sistemi (mockup'taki tam akış)
  • Çapraz Modül 1: Beslenme — diyetisyen rolü + plan + karne entegrasyonu
  • Sertifikasyon takibi

Faz 3 — Uzmanlaşma (3-4 ay)

M2Kalecilik — kaleci antrenörü rolü + ayrı drill seti + karne

M3Sezon Dışı Hazırlık — yıllık plan + kondisyoner ekosistemi

M4Return to Play — sakatlık kayıt + 5 fazlı protokol + concussion özel akış + uzman ekip işbirliği

M5Pozisyon-Özel Drill'ler — pozisyon profili + maç analizi

M6Futsal/İç Saha — sezon takvim + hava acil durum akışı + futsal karne

  • TFF entegrasyonu
  • Sigorta entegrasyonu
  • Multi-akademi (white-label) altyapısı

Faz 4 — Olgunluk (sürekli)

  • Yapay zeka destekli karne yorumu (LLM ile özet, ASLA otomatik karar değil — KVKK m.11)
  • Veli app yapay zeka koçu (sınırlı tavsiye)
  • Pediatrik veri seti üzerinden anonim performans karşılaştırma
  • ISO 27001 sertifikasyonu
